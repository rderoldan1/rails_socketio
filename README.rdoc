## Running it


You need to start 2 processes, the rails server and de socket.io server:

1. In one terminal run `rails s`
1. In other terminal start the socket.io server, it is under lib/socket_server run `nodemon ib/socket_server/server.js`

## Socket.io

The code is under lib/socket_server

## Prerequisites

1. Ruby on Rails
2. NodeJS
3. NPM
4. Redis
